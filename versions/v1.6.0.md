# v1.6.0 - Gestão de Tempos Livres (Férias e Licenças)

## Resumo
Esta versão foca na implementação do gerenciamento de períodos de férias e licenças longas. Agora é possível definir intervalos de datas que serão considerados dias sem aula, impactando o cálculo de datas finais e métricas.

## Novas Funcionalidades
### 1. Sistema de Férias e Licenças
- **Visualização**: Novos dias marcados com a cor amarela (`bg-yellow-200`) no calendário.
- **Inserção por Período**: Interface dedicada para adicionar intervalos (Data Início e Data Fim) com nome personalizado.
- **Listagem e Gestão**: Nova seção na barra lateral listando os períodos cadastrados, permitindo removê-los facilmente.

### 2. Cálculos Aprimorados
- **Previsão de Término**: A lógica de cálculo automático do fim do semestre (`calculateEndDate`) agora ignora corretamente os dias dentro dos períodos de férias/licença.
- **Métricas de Aula**: Os dias de férias são excluídos da contagem de dias efetivos de aula.

### 3. Melhorias na Interface
- **Formulários Expandidos**: O painel de controle (`CalendarControls`) agora inclui seções distintas para Feriados, Recessos (dia único) e Férias (período).
- **Legenda Atualizada**: Item "Férias/Licença" adicionado à legenda interativa.

## Detalhes Técnicos
- **Novos Estados**: `vacationPeriods` (array de objetos com start/end/name).
- **Hooks de Otimização**: `vacationDays` (Set) e `vacationNames` (Map) são derivados via `useMemo` para performance eficiente no rendering do grid.
- **Persistência de Dados**: O arquivo JSON exportado agora inclui o array `vacationPeriods`, garantindo que os dados sejam salvos e restaurados corretamente.

## Correções
- Ajuste na estrutura do JSON importado para suportar os novos campos sem quebrar compatibilidade com arquivos antigos.
