# v1.9.0 - Multi-UCs por Dia

## Resumo
Introduz a capacidade avançada de agendar até 4 Unidades Curriculares (ou diferentes tipos de aula) no mesmo dia. O sistema visualiza essa sobreposição dividindo a célula do dia em quadrantes coloridos, tanto na interface web quanto no PDF Compacto.

## Mudanças
### Interface Web
- **Seleção de Cores**: Ao selecionar uma cor (UC) para um dia que já possui uma cor, o sistema agora **adiciona** a nova cor em vez de substituir.
- **Toggle**: Clicar na mesma cor novamente a remove.
- **Visualização**:
    - 1 Cor: Preenchimento total.
    - 2 Cores: Divisão vertical (50% / 50%).
    - 3 Cores: Divisão avançada (1 Topo, 2 Baixo).
    - 4 Cores: Grade 2x2.
- **Limite**: Máximo de 4 cores por dia.

### Exportação PDF
- **PDF Compacto**: A lógica de desenho vetorial (jsPDF) foi atualizada para replicar fielmente a divisão de células, garantindo que o calendário impresso reflita as múltiplas UCs.

## Como Usar
1.  Selecione uma UC na legenda (ou cor).
2.  Clique no dia desejado.
3.  Selecione outra UC.
4.  Clique no mesmo dia. O dia será dividido.
