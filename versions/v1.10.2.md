# v1.10.2 - Correção de Importação e Sincronização de Estado

## Resumo
Correção técnica para garantir que a importação de arquivos JSON sobrescreva corretamente todos os campos da interface, especialmente as configurações gerais.

## Mudanças

### Sincronização de UI
- **Componentes Controlados**: Adição da prop `value` aos inputs de "Início do Semestre" e "Fim do Semestre" no componente `CalendarControls`.
  - Anteriormente, estes campos eram "semi-controlados" (tinham `onChange` mas não `value`), o que impedia que refletissem mudanças de estado vindas de fontes externas como a importação de arquivos.

### Robustez na Importação JSON
- **Reset de Estado**: A função `handleImportJson` foi refatorada para garantir que, se um campo estiver ausente no arquivo JSON carregado, o estado correspondente seja resetado para seu valor inicial (null, vazio ou padrão).
  - Isso evita que dados de um projeto anterior permaneçam "sujando" as configurações gerais após carregar um novo arquivo.
- **Campos Afetados**:
  - Nome da Turma
  - Datas de Início/Fim
  - Feriados, Reposições e Recessos (incluindo nomes)
  - Unidades Curriculares
  - Períodos de Férias/Licenças
  - Configurações de Carga Horária e Auto-cálculo
