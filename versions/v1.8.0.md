# v1.8.0 - Personalização de Carga Horária por UC

## Resumo
Adiciona a flexibilidade de definir uma carga horária diária específica para cada Unidade Curricular, permitindo que UCs com intensidades diferentes (ex: 4h/dia vs 8h/dia) tenham suas datas de término calculadas corretamente.

## Mudanças
### Funcionalidades
- **Input de Horas por Dia**: Novo campo "Horas/Dia (Opcional)" no formulário de inclusão/edição de UC.
- **Cálculo Híbrido**: Se o campo estiver preenchido, o sistema usa o valor da UC; caso contrário, usa o valor padrão global do curso.
- **Correção de Cálculo**: O algoritmo de previsão de término agora considera corretamente os dias de **Recesso** e **Férias** cadastrados, que antes eram ignorados neste cálculo específico.

## Benefícios
- Maior precisão no planejamento de cursos modulares com cargas horárias variáveis.
- Automatização mais confiável das datas de término.
